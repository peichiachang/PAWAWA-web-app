# 體積計算誤差分析報告

## 問題描述
- **實際容量**：2000ml (2L)
- **計算結果**：3817ml
- **誤差**：1817ml (91%)

## 計算結果分析

根據測試，3817ml 最接近以下輸入組合的計算結果：

### 最可能的輸入值
- **圓柱形**：直徑 22cm，高度 10cm → **3801ml** (誤差僅 16ml)
- **圓台形**：頂直徑 30cm，底直徑 25cm，高度 6cm → **3574ml** (誤差 243ml)

## 2L 圓柱形容器的合理尺寸

如果實際是圓柱形 2L 容器，合理的尺寸應該是：

| 高度 | 直徑 | 計算體積 |
|------|------|----------|
| 15cm | 13.0cm | 1991ml ✓ |
| 20cm | 11.3cm | 2000ml ✓ |
| 25cm | 10.1cm | 2000ml ✓ |

## 誤差原因分析

### 1. **輸入值過大**
如果輸入的是「直徑 22cm，高度 10cm」：
- 計算結果：3801ml
- 但實際容量只有 2000ml
- **可能原因**：
  - 測量的是**外徑**而非**內徑**（飲水機有厚度）
  - 測量的是**最大寬度**而非**有效容量直徑**
  - 測量位置不正確

### 2. **形狀選擇錯誤**
如果實際是圓柱形，但選擇了圓台形狀：
- 圓台公式會產生更大的體積
- 特別是當頂/底直徑差異不大時，圓台體積會比圓柱大

### 3. **測量誤差**
- 飲水機通常有**不規則形狀**（非完美圓柱）
- 內部可能有**結構件**（水泵、過濾器等）佔據空間
- **實際可用容量** < **幾何體積**

## 圓台體積公式說明

圓台（截頭圓錐）體積公式：
```
V = (π × h / 3) × (R² + R×r + r²)
```
其中：
- R = 頂半徑
- r = 底半徑  
- h = 高度

**重要特性**：
- 當 R = r（圓柱）時，公式簡化為：`V = π × r² × h`
- 當 R ≠ r 時，體積會**大於**相同高度的圓柱

## 解決方案

### 方案 1：直接輸入已知容量（推薦）
- 選擇「已知容量」輸入方式
- 直接輸入 `2000`（毫升）
- 避免測量和計算誤差

### 方案 2：正確測量內徑
- 測量**內部有效容量**的直徑
- 考慮內部結構件佔據的空間
- 使用**內徑**而非外徑

### 方案 3：使用水容量測試
- 實際倒水測試容量
- 記錄實際可裝水量
- 使用測試結果作為已知容量

## 建議

對於**飲水機等大型容器**：
1. ✅ **優先使用「已知容量」輸入方式**
2. ✅ 容器標籤通常會標示容量（如 2L、2000ml）
3. ✅ 比測量尺寸更準確、更簡單

對於**食碗等小型容器**：
1. ✅ 使用「測量尺寸」輸入方式
2. ✅ 測量內徑和高度
3. ✅ 注意單位（公分，不是毫米）

## 計算公式驗證

所有計算公式都是標準的幾何體積公式，**公式本身沒有錯誤**。

誤差主要來自：
- **輸入值的準確性**
- **形狀選擇的適配性**
- **實際容器與理想幾何形狀的差異**
